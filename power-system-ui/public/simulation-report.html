7. 엔지니어링 발견 사항 및 권고 (EPC Design Review)

3 Critical · 4 Warning · 5 Info
🔴 Critical #1 — PP-101 기동방식 변경 필수

PP-101 (1,500kW, 22.9kV) DOL 기동 시 전압강하 17.4%로 IEC 60034-1 권고치(15%)를 초과합니다. 동시에 22.9kV 모선에 연결된 다른 모터 및 제어계통의 오동작 위험이 있습니다.
권고: DOL → Star-Delta 또는 Soft-Starter로 변경. Star-Delta 적용 시 기동전류 1/3 → 전압강하 ~5.8% 예상. 공정 측 토크요구사항(≥85% full-load torque at start) 확인 후 결정.

🔴 Critical #2 — 0.38kV 버스 전압강하 경고

0.38kV MCC-A 버스 전압이 0.9714 pu (369 V)로 정격 380V 대비 2.86% 낮습니다. PP-201 모터 케이블 전압강하까지 합산하면 PP-201 단자 전압은 0.9429 pu (358 V) — IEC 60034 권고 ±10% 이내이나 운전 마진이 매우 좁습니다.
권고: TR-MCC1 탭 위치를 tap_pos = +1 (0.38→0.38+2.5%=0.3895 kV)로 조정하여 MCC 버스 전압을 약 1.0 pu로 올림.

🔴 Critical #3 — 154kV 계통연계 Breaker 용량 검토

154kV Main Bus의 Ik'' = 18.73 kA, Ip = 47.6 kA입니다. 현재 설정된 CB-HV 정격 40kA 는 Ik'' 기준으로는 PASS이나 Peak Current Ip = 47.6 kA > CB making capacity 추정값일 수 있습니다. IEC 62271-100 §4.102: Icm ≥ 2.5 × √2 × Ik'' = 66.2 kA 요구.
권고: 154kV GIS/AIS CB의 Making capacity (Icm) 를 카탈로그에서 재확인. 일반적으로 최소 80 kA 이상 제품 선정 필요.

⚠ Warning #4 — TR-MCC1 보호 협조 검토 필요

TR-MCC1 (2 MVA) 2차 측 단락전류 22.8 kA에 대해 CB-201~CB-203 (MCCB, 36kA) 은 용량 충족이나, CB-103 (VCB, 25kA)의 1차 측 단락전류 환산값 = 22.8 × (0.38/22.9) = 0.378 kA 이므로 VCB 통과전류 기준은 문제없음. 단, 계전기 협조 시간 마진이 CB-103 (TMS=0.2) → CB-201 (TMS=0.1) 간 0.3초 이상 확보 여부 TCC Viewer로 확인 권고.

⚠ Warning #5 — 0.38kV 부스바 열·기계 강도

0.38kV MCC-A 부스바 최대 피크전류 Ip = 57.9 kA. IEC 60865-1에 따른 부스바 기계강도 검토 필요. 일반 표준 MCC 부스바 (63kA 1초 열강도 등급) 적용 시 통상 만족하나, 도면에 부스바 규격(mm²), 지지간격(mm), Icw 1s 명시 필요.

⚠ Warning #6 — PP-201 케이블 전압강하 누적

MCC 버스 2.86% + LV-F201 2.91% + F201-Motor 2.75% = 누적 전압강하 8.52% (PP-201 단자기준). IEC 60364-5-52 권고치 5% 초과. 케이블 단면적 증가 또는 TR 탭 조정 병행 필요.

⚠ Warning #7 — 무효전력 보상 미적용

전체 시스템 역률: P = 5.23 MW, Q = 3.07 Mvar → PF = 0.863. 한전 역률 기준 0.95 미만 시 전기요금 할증(역률요금) 적용. 22.9kV 모선에 1,200~1,500 kvar 콘덴서 설치 검토 (PF 개선 후 약 0.95).

ℹ Info #8 — 비상발전기 용량 검토

G-EMG (1,500 kVA) 비상전원 공급 대상 부하를 정의하지 않았습니다. 일반적으로 EPC 프로젝트에서 비상부하 = 안전계장계통 + 비상조명 + 소화펌프. 비상부하 목록(Essential Load List) 작성 후 용량 재검토 권고.

ℹ Info #9 — CM-101 VFD 고조파

CM-101 (2,000kW VFD)은 고조파 전류원입니다. IEEE 519 기준 22.9kV 버스 THDv 한계 5%. VFD 사양에서 h5=15%, h7=8%, h11=5%, h13=4% 수준이 일반적. 고조파 해석(Harmonics Analysis) 별도 수행 권고. 필요 시 12-pulse 구성 또는 Active Filter 검토.

ℹ Info #10 — Arc Flash 분석 미수행

22.9kV 및 0.38kV 버스에 대한 Arc Flash 분석 (IEEE 1584)이 수행되지 않았습니다. 단락전류 입력 후 소프트웨어 내 Arc Flash 버튼으로 Incident Energy 및 PPE Category 산출 후 도면에 반영 필요 (IEC 60900, NFPA 70E 준수).

✓ OK #11 — MV 차단기 용량 모두 적합

22.9kV 버스 Ik'' = 9.24 kA < CB-101/102/103 정격 25 kA. 여유율 +171%. Ip = 23.4 kA < Making capacity 62.5 kA. 현재 CB 규격은 적합합니다.

✓ OK #12 — TR-MAIN 부하율 양호

TR-MAIN 부하율 17.6% — 미래 부하 증설(Future Load) 포함 시에도 여유 충분. 플랜트 확장 시 동일 변압기 유지 가능 (최대 증설 허용: +35MW 추가).

9. PowerFlow Analyzer 사용 소감 (엔지니어 피드백)

Tool Feedback

✅ 잘 된 점

Bus 전압 결과가 캔버스에 직접 오버레이되어 시각적으로 직관적.

Motor List Excel Import → 자동 SLD 작성 기능으로 MCC 설계 시간 대폭 단축.

Load Flow + Short Circuit + Arc Flash + Cable Sizing 통합 실행 편리.

Undo/Redo, 인라인 이름 편집, Multi-select 삭제 — ETAP 대비 빠른 편집 UX.

❌ 개선 필요

단상 고장 해석 없음: IEC 60909 지락 고장(1LL, 2LL) 계산 미지원. EPC 프로젝트에서는 지락 고장 전류 계산이 보호 설정의 핵심.

케이블 데이터베이스 빈약: 국내 표준(KS C IEC 60502) 케이블 규격 DB 없음. 실무에서는 XLPE/CV 케이블 선정이 대부분.

변압기 탭 자동 최적화 없음: MCC 버스 전압강하 해결을 위한 탭 위치 자동 계산 기능 필요.

역률 보상 계산 없음: 콘덴서 용량 자동 계산 기능 없음 (전력요금 최적화 핵심 기능).

Load Schedule 없음: 부하 목록표(Load List)에서 도면으로 직접 연동되는 기능 필요. Motor List Import는 시작점이나 Load/Generator도 지원 필요.

결과 비교 기능 없음: 베이스 케이스 vs 설계 변경 케이스 비교 (ETAP 'Study Case' 개념). 탭 변경 전/후 비교 등.
